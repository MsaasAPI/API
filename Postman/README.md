Postman is a popular API testing tool, and it is no exception for MSaaS Case API client implementation. Experience suggests that extra time investment in Postman can familiarize partner with the MSaaS Case API, flatten the learning curve, alleviate confusion, and facilitate trouble-shooting. Although the use of Postman is totally optional, it is nevertheless our hope that the readers will find Postman helpful in the on-boarding process. The intended audience of this step-by-step guide is the web developers.
